# FacilePlan

FacilePlan è un’app iOS per la gestione di calendario e attività.

## Descrizione
FacilePlan permette di organizzare eventi e attività con un’interfaccia nativa iOS semplice e veloce.  
L’app può collegarsi a servizi esterni configurati dall’utente per recuperare dati.

## Privacy
- Nessun dato viene raccolto
- Nessun tracking o analytics
- Nessuna pubblicità
- Tutti i dati restano sul dispositivo dell’utente
- Le credenziali vengono salvate in modo sicuro tramite Keychain iOS
